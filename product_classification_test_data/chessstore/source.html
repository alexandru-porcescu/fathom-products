  • Product Description

    Picture this elegant chess set sitting in your home looking beautiful and sitting ready for a game of chess. This luxurious chess set from the workshop of Italfama in Florence, Italy and is one of their most impressive sets. This set is made from solid brass and features a contemporary Staunton design with a graceful profile and detailed knights. This is a large chess set with a 3.375” tall king while the chess board has 2” squares. This package includes our Walnut and Maple Chess Case. The rich golden-brown colors of the walnut creates a beautiful backdrop for the brass and nickel chess pieces. There are 2 drawers, one on each player’s side for storing the pieces if you don’t want them sitting out. Each drawer has 16 individual compartments that are lined with a beige velour material for conveniently storing the pieces and keeping them separated and protected. Whether you are looking for a special chess set for yourself or a special gift, you will not find a more beautiful chess set!

    Chess Pieces:

    • King Height: 3.375"
    • King Base: 1.125" 
    • Material: Brass & Nickel Plated Cast Zinc 
    • Chess Set Weight: 8 lbs. 
    • Felt Pads 
    • Comes packaged in Italfama green and gold fancy cardboard box.
    Chess Case:
    • Material: Walnut & Maple Veneer Over Wood Composite
    • Finish: Satin Polyurethane
    • Dimensions: 18.75" x 18.75" x 3.25"
    • Square Sizse: 2"
    • Compartment Dimensions:
      • Large (8 each drawer): 4" x 1.7" x 1.7"
      • Small (8 each drawer): 3.5" x 1.7" x 1.7"
